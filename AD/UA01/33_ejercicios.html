3.3. Ejercicios

Tarea 1.1. La carpeta de ejercicios

Apartado A

Vamos a crear la estructura de carpetas donde guardaremos nuestros ejercicios.
La ubicación principal será la Carpeta personal siguiendo la ruta C:\Users\[nombre]\ donde nombre es vuestra carpeta personal.

Para realizar esta tarea, una aplicación en Java recorrerá el documento de texto adjunto y creará la estructura siguiendo su patrón.

Deberás entregar los siguientes archivos:

  1. Código fuente en java
  2. Captura, mediante Recortes, de la estructura de carpetas generada. Usa TREE en la línea de comandos para mostrarla.

Entrega ambos documentos en la misma tarea de Moodle.

Apartado B

En este ejercicio, vamos a automatizar la creación de un sitio web.

Deberás recorrer la estructura de carpetas anterior e insertar un fichero .html, con la estructura básica de una página Web, en cada una de ellas.

Estos documentos HTML deberán contener el siguiente código:

<html>
   <head>
      <title> Nombre de la carpeta </title>
   </head>
   <body>
      <h1>Nombre de la carpeta</h1>
      <h3>Autor: [nombre_del_alumno]</h3>
   </body>
</html>

Deberás entregar los siguientes documentos:

  1. Código fuente en Java
  2. Archivo comprimido ZIP de la estructura generada

Súbelos a la misma tarea en Moodle.
