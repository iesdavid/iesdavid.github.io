UD3. SQL: SELECT

¿Para qué utilizamos los comandos SQL? El uso común es la selección de datos desde tablas ubicadas en una base de datos para obtener una nueva tabla de datos que será almacenada posteriormente o no.

Inmediatamente, vemos dos palabras claves: necesitamos seleccionar (SELECT) la información desde (FROM) una tabla. La tabla no es más que el contenedor que reside en la base de datos donde se almacena la información. Por lo tanto tenemos la estructura SQL más básica:

SELECT "nombre_columna" FROM "nombre_tabla";

Para ilustrar el ejemplo anterior, suponga que tenemos la siguiente tabla Info_tiendas:

Nombre_Tienda Ventas Fecha
Utrera 1500 05-Ene-1999
Gines 250 07-Ene-1999
Aznalcázar 300 08-Ene-1999
Bormujos 700 08-Ene-1999 
Tabla Info_tiendas

Para seleccionar todos los negocios en esta tabla, ingresamos,

SELECT Nombre_Tienda FROM Info_tiendas;

Resultado:

Nombre_Tienda
Utrera
Gines
Aznalcázar
Bormujos

Como veremos posteriormente, pueden seleccionarse los nombres de columnas múltiples separadas por comas; así como también los nombres de tablas múltiples, también separadas por comas, tras la cláusula FROM.

Del inglés, SELECCIONA. Permite obtener de la base de datos lo indicado a continuación.

Del inglés, DESDE. Indica de dónde obtener los datos indicados en el SELECT. Puede ser de una tabla, varias e incluso otro SELECT.
