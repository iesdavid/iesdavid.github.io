7.2.2. Ejercicio

XML como Base de Datos

Crea un archivo XML para almacenar los datos de una tienda.

Se deberá recoger el mayor número de datos posible de cada elemento (no el mayor número de elementos). Para ello, estructura el archivo XML en las etiquetas y atributos necesarios y crea el documento DTD que impida almacenar información no deseada.
