5.4.2. Ejercicios interactivos online de H5P

¿Qué puede hacer H5P por mis alumnos?

H5p es una plataforma de creación de contenidos interactivos, gratuita y abierta, con todas las ventajas que proporciona el software libre en educación, ampliando las posibilidades de aprendizaje de nuestros alumnos.

Lola Alberdi

 

Según Lola Alberdi, desde la página del CEDEC (enlace a nueva ventana) H5P es:

  • multiplataforma (funciona el Linux, Windows, IOS),
  • de código abierto y por lo tanto sostenible en el tiempo, asegurando la perdurabilidad de nuestras creaciones,
  • con libertad para usar, copiar, modificar y distribuir el sofware,
  • optimiza recursos, reduciendo el costos de equipos,
  • crea alumnos libres, no dependientes de un producto concreto ya que se enseña a trabajar con una tecnología.

Para comenzar a utilizar la herramienta, podemos consultar un sencillo tutorial de usuario realizado por Iván Otero González de la Universidad de Vigo y responsable del proyecto Up2U que conecta la Enseñanza Superior con las enseñanzas medias:

Alojamiento de las actividades H5P del Proyecto REA Andalucía

Todos los recursos interactivos H5P generados en el marco del Proyecto REA Andalucía, serán catalogados y alojados en el blog Ejercicios REA, perteneciente al espacio Blogs Averroes de la Consejería de Educación de la Junta de Andalucía. 

A partir de aquí se obtendrá el código embed para incorporar el recurso al .elp del REA correspondiente. 

Se puede realizar búsqueda de los recursos por: 

  • Materia
  • Etapa
  • Curso
  • Bloque de contenidos

Drag the words

Esta actividad se podría realizar con un idevice de eXeLearning y ser usada sin conexión

Arrastra el tipo de palabras a su lugar correcto. Me tienes que mandar las frases y la palabra oculta entre *

Single Choice Set

Esta actividad se podría realizar con un idevice de eXeLearning y ser usada sin conexión

Permite múltiples preguntas en las que sólo una opción es correcta. (Selección múltiple)

Fill in the blanks

Esta actividad se podría realizar con un idevice de eXeLearning y ser usada sin conexión

Multiple Choice

Esta actividad se podría realizar con un idevice de eXeLearning y ser usada sin conexión. Además en eXeLearning permite incorporar varias preguntas.

Image Sequencing

Si sólo es texto se puede realizar con un idevice y no hace falta conexión.

Para implementarla hacen falta las imágenes y el orden, así como, en su caso, los archivos de audio de ayuda.

Poner en orden una secuencia de imágenes.

Dictado

Para implementarla hacen falta las frases y los archivos de audio de cada una de ellas.

En este ejemplo los audios se han creado con: https://notevibes.com/cabinet.php

Mark the word

Ejercicio para encontrar tipos de palabras. Tienes que enviarme el texto con las palabras que deben de señalar entre *

Essai

El alumno/a va a redactar. La única condición y lo que evalúa es que incluya determinadas palabras. Se puede puntuar por el número de veces que aparece en el texto.

Find the word

Sopa de letras. Adverbios de modo. Me hace falta la lista de palabras a buscar separadas por coma

Interactive Vídeo

Esta opción podría hacerse como idevice y ser usada sin conexión.

Vídeo con actividades interactivas, me tienes que mandar el vídeo y los tipos de actividades que pueden ser:

Texto, imágenes, hiperenlaces, tablas, numeración y viñetas (para resaltar algún contenido o resumir), preguntas, rellenar huecos, single chioce, multiple choice, de todas ellas debes decirme qué pregunta es, cual es la correcta, y las opciones a poner.

Memory game

Une las imágenes, si quieres puedes poner una pista de audio que suena cuando le das la vuelta a la tarjeta (es opcional). Me tienes que mandar las imágenes a emparejar y si quieres pistas, los audios (https://notevibes.com/cabinet.php). Si quieres usar letreros estos están hechos con una presentación (un nombre por dispositiva) y luego está exportado a jpg.

Flashcards

Son diferentes tarjetas, cada una de ellas con una pregunta. Me hacen falta las preguntas y las respuestas y la imagen que quieres que tenga. Puede tener cada tarjeta una imagen si la pregunta es sobre una imagen o una estándar (decorativa) creada como nube de palabras, por ejemplo, si son preguntas de texto

True/False Question

Es una única pregunta para un vídeo o una imagen

Find the hotspot

Creamos una imagen con varios puntos calientes, uno de ellos es la respuesta a la pregunta

planteada

Agamotto

Imágenes y texto relacionadas.

Debes darme las imágenes y el texto asociados a ellas.

Find Multiple Hotspots

Elegir varios puntos calientes en una imagen, debes darme la imagen con las marcas delas opciones verdaderas en ellas.

Summary

Debes darme una serie de oraciones y sólo una de ellas es la buena.

Línea de tiempo

Debes darme: Los periodos y el título y el texto que quieres que aparezcan en cada pedriodo.

Graba una poesía

Guess the answer

Este ejercicio consiste en adivinar la respuesta a partir de la imagen, pero no se debe introducir ninguna información.

Collage

Esta actividad tampoco requiere que el alumno/a introduzca ninguna información, simplemente permite combinar varias imágenes en forma de collage con diferente composición.

Arrastra

Esta actividad consiste en relacionar un texto que debe relacionarse con una parte de la imagen.

Personality quiz

Esta actividad permite realizar un cuestionario de personalidad, a través de diferentes preguntas que clasificarán la personalidad en función de las respuestas seleccionadas.

Advent Calendar

Este componente permite crear un calendario de adviento para tratar un contenido diario. El contenido a incluir puede ser una imagen, un texto, un archivo de audio o vídeo o un enlace.

Image juxtaposition

Más que una actividad, es una combinación de dos imágenes que permite de forma dinámica cambiar de una a otra. Quizás tenga más utilidad en otras materias, aunque visualmente resulta bastante atractiva.

Image Pairing

Unir imágenes relacionadas arrastrando unas con otras.

Course presentation

Presentación que le permite al docente añadir interacciones de opción múltiple, llenar los huecos, textos y otros tipos a sus presentaciones:

Debes entregarme la presentación en formato ppt, o impress,  con las diapositivas que quieras que aparezcan, y las actividades interactivas con las preguntas y las opciones a elegir y la que es correcta. 

Accordion

Definición de los tipos de palabras. Me tienes que mandar los conceptos y la definición. 

Branching Scenario

Se trata de un juego de pistas.  Me hace falta la historia y cada una de las pistas. 

Columna

Excelente para hacer comentarios de texto o diferentes tipos de ejercicio que parten de una lectura o de un vídeo

Question Set

Para hacer una prueba con varios tipos de preguntas (Siempre texto)

Documentation Tool

Herramienta para ayudar a documentar un proyecto, partiendo de una serie de pasos: definir el proyecto, los Objetivos, la consecución de esos objevios, etc...

Al final del proceso genera la documentación en un documento de Word.

Speak the word set

Conjunto de actividades "speak the word" (grabación de audio). Al finalizar da la calificación conjunta de todos los ejercicios de audio realizado.

Interactive book

Este componente permite crear diferentes capítulos con contenido basado en otros componentes H5P, de esta forma se puede incluir varios capítulos con una información y su correspondiente actividad.

Virtual Tour (360)

Este componente permite realizar una visita virtual a través de imágenes 360º, insertando puntos de información y encadenando unas imágenes con otras. No le hemos encontrado aplicabilidad para las materias de Lengua.

KewAr Code

Este componente permite crear un código QR de una URL.
