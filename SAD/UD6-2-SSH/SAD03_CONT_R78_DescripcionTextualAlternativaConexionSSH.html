Descripción textual alternativa del vídeo "Conexión SSH"

En el vídeo se muestra el comando SSH que permite realizar una conexión, como usuario arubio, desde un equipo cliente al servidor cuya IP es 192.168.1.53.

Al ser la primera vez que se intenta realizar dicha conexión, nos avisa que la clave pública del servidor no está en nuestro anillo de confianza, con lo que no se puede garantizar la autenticidad del servidor, preguntando si deseamos continuar con el intento de conexión. Contestamos yes, con lo que se muestra un mensaje indicando que se añadirá la IP del host remoto, es decir, del servidor, a la lista de host de confianza del equipo cliente.

A continuación, se solicita la password del usuario remoto arubio, se introduce correctamente dicha password y se establece la conexión. Una vez conectados al servidor, se ejecuta el comando ifconfig para mostrar todas las interfaces de red existentes en el equipo remoto. Además, se ejecuta el comando whoami que nos confirma que estamos conectados con el usuario arubio. Seguidamente, se ejecuta el comando exit que permite finalizar la conexión SSH.

Finalmente, y una vez cerrada la conexión, se muestra el contenido del fichero known_host en cuya segunda entrada se puede apreciar:

  • La identificación del equipo remoto al que se acaba de realizar la conexión.
  • El algoritmo de cifrado empleado.
  • La clave pública del equipo remoto.
