3.3. Interfaces de usuario

El SDK de Android nos brinda las clases necesarias para acceder a la interfaz de usuario del sistema operativo Android de forma sencilla. Podemos heredar de esas clases para crear nuestra interfaz de usuario a nuestro gusto, para que termine siendo lo más parecida al mockup que creamos en la fase de análisis.

En este punto, vamos a estudiar las clases más comunes para la construcción de una interfaz de usuario de una app Android.
